# Sacred Dominion - Phase 2 Test Checklist

Phase 2 currently adds base records only.

No gameplay effects are active yet.

---

## Records to check

Beliefs:

- Sacred Administration / Heilige Verwaltung
- Charitable Works / Armenfuersorge
- Pilgrimage Network / Pilgernetzwerk

Policies:

- Tithe Administration / Zehntverwaltung
- Order Patronage / Ordensfoerderung

---

## Test steps

1. Copy the mod folder into the local Civilization VI Mods folder.
2. Enable Sacred Dominion in Additional Content.
3. Start a new game.
4. Check that the game starts without database problems.
5. Check that the new text entries are not shown as missing localization keys.

---

## Logs

Check the local Civ VI log folder after starting the game.

Most important files:

- Database.log
- Localization.log
- Modding.log

---

## Pass condition

The Phase 2 record test passes when:

- the mod can be enabled
- a new game starts
- the new records do not create database errors
- the English and German text keys load correctly

---

## Next step after pass

Activate only the first small effect for Tithe Administration.

Then test again before adding the next effect.
