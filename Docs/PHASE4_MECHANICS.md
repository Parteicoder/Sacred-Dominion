# Sacred Dominion - Phase 4 Mechanics

Phase 4 adds the first sacred decrees.

Because custom UI is not started yet, decrees are simulated as special policy cards.

---

## Design boundary

Use simple Civ VI systems first:

- policy records
- small yield modifiers
- existing localization keys
- no custom UI
- no Lua state system

---

## First decree pass

## Declare Tithe

German: Zehnt ausrufen

Current direction:

- economic faith administration
- small Gold support from religious infrastructure

## Holy Festivals

German: Heilige Feste

Current direction:

- culture and faith celebration
- small Culture support from religious infrastructure

## Missionary Mandate

German: Missionsmandat

Current direction:

- missionary push
- small Faith support from religious infrastructure

## Mobilize Orders

German: Orden mobilisieren

Current direction:

- order mobilization
- small Faith and Gold support

## Wonder Veneration

German: Wunderverehrung

Current direction:

- wonder and sacred site veneration
- small Culture or Faith support

---

## Rule for first pass

The first pass should be conservative.

No loyalty snowball. No tourism spike. No cost reduction until local testing confirms the database setup is clean.
